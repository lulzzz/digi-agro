Backend:
- install: Java, Gradle, PostgreSQL
- create data base with name: digi-agro-db
- run Application.java

Frontend:
- install: Node JS
- install Typings, execute: npm install -g typings
- install Angular CLI, execute: npm install -g @angular/cli@latest
- install dependencies, execute: npm install
- run application, execute: ng serve
- open browser: http://localhost:4200
